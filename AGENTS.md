# OpenClaw Agent Guide

## YOU ARE INSIDE A DEVCONTAINER
You are running inside a VSCode devcontainer. You can check the configuration of the container you are running in at `.devcontainer/docker-compose.workspace.yml`
You have no option to rebuild this container, that you are running in. You can not alter it in any way, as the user has to click the rebuild option in VSCode. A Rebuild should not be neccessary, as it is NOT YOUR TASK to edit the devcontainer setup. Your task is only to edit the other files and folders. All files and folders including the .devcontnainer folder are at `/home/vscode/workspace`

##  Purpose and Task
This Project is a setup to run OpenClaw in a complete local mode. This setup is developed by an opencode agent. You are this opencode agent. Develop the setup by editing files in this workspace, and use `docker` commands and headless browsers to verify your changes.
